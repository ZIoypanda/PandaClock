## PandaClock 1.0.3

- Окно "Доступна новая версия" теперь показывается до запуска часов.
- Если обновление найдено, часы открываются только после выбора "Позже" или после установки обновления.
- В контекстное меню часов по ПКМ добавлена кнопка "Закрыть".
- Из стандартного меню трея убраны лишние пункты управления, которые дублировали панель настроек.
- Release теперь публикуется с отдельными patch notes, которые отображаются в "Что нового?".
